# ✨ JoinedQuitMS
A simple plugin to replace boring default join and quit messages with your custom messages.
# ⚙️ Configuration (config.yml)
```yaml
join-message: "&a[+] &f{player}"
quit-message: "&4[-] &f{player}"
```
